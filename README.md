# Business Finance Quest: The Financial Empire

Interactive formative HTML game for Business Finance Week 1.

## Files
- `index.html` - main game
- `sounds/` - background music and sound effects

## GitHub Pages Setup
1. Upload all files in this folder to your GitHub repo:
   https://github.com/Oblivion08/BFinanceWk1
2. Go to Settings > Pages.
3. Source: Deploy from a branch.
4. Branch: main.
5. Folder: /root.
6. Save.

Live site:
https://oblivion08.github.io/BFinanceWk1/
